Diagonal Dominoes

Diagonal Dominoes is based on a mini-game found in the PS2 video game 'Avatar: the Last Airbender'. Each player begins with five two-toned tiles, as depicted here.

The human player (Player 1) can start the game with any tile in their collection. From there, each player must put down a tile whose color corresponds with the open tile on the board.

Once the human begins the game, the computer creates a tree of all possible moves. From there, if a path exists that can guarantee the computer's victory, it will take it. If not, the computer will opt for the path that has the most opportunity for human error, as this ups its chances of winning.

The game ends either when a player has no more tiles with the right color to play, or no more tiles at all (this case will only be available to the human player, as they always go first).
